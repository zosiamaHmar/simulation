Step 1 · The Scene
Setting Up the Geometry

Let P be the observer, 60 m above the lake surface.

Let C be the cloud at height h metres above the lake.

Let C' be the reflection of the cloud in the lake.

Because the lake is a mirror: depth of reflection = height of cloud

So C' is h metres below the lake surface.

Key Distances from Observer P
Vertical distance to cloud: h − 60 m
Vertical distance to reflection: h + 60 m
Both share the same horizontal distance d
👁 Show both triangles on canvas →
Step 2 · The Key Insight
Two Right Triangles, One Unknown

From observer P, draw a horizontal line. Two right triangles are formed:

Triangle 1: Elevation to Cloud (30°)
tan 30° = (h − 60) / d
1/√3 = (h − 60) / d
d = (h − 60)√3 ......(1)
Triangle 2: Depression to Reflection (60°)
tan 60° = (h + 60) / d
√3 = (h + 60) / d
d = (h + 60) / √3 ......(2)
💡 Why is the reflection distance h + 60? Click to see.
The reflection is h metres BELOW the lake (same as cloud height). The observer is 60m ABOVE the lake. So total vertical separation = h + 60.
Step 3 · The Calculation
Solving for Cloud Height

From equations (1) and (2), both equal d:

Set the two expressions for d equal
(h − 60)√3 = (h + 60) / √3

Multiply both sides by √3:

(h − 60) × 3 = h + 60
3h − 180 = h + 60
3h − h = 60 + 180
2h = 240

Therefore:

Final Answer
h = 120 metres
The cloud is 120 m above the lake surface
Do NOT add the angles (30° + 60° = 90°). These are two separate measurements from the same observer position!

Phase 5 · Master
Three Variants to Test Yourself
Variant 1 · Different Numbers

Observer is 30m above lake. Elevation = 45°, Depression to reflection = 60°. Find cloud height.

💡 Show solution
d = (h−30)tan45° = (h+30)/tan60° → h−30 = (h+30)/√3 → h√3 − 30√3 = h + 30 → h(√3−1) = 30(√3+1) → h = 30(√3+1)/(√3−1) = 30(2+√3) ≈ 111.96 m
Variant 2 · General Formula

Prove that for observer height h, elevation angle α, depression angle β: cloud height H = h(tan β + tan α)/(tan β − tan α)

💡 Show derivation
d = (H−h)/tanα = (H+h)/tanβ → (H−h)tanβ = (H+h)tanα → H(tanβ−tanα) = h(tanβ+tanα) → H = h(tanβ+tanα)/(tanβ−tanα)
Variant 3 · Real World

A drone at 100m altitude spots a mountain peak at 30° elevation and its reflection in a lake at 45° depression. How high is the peak?

💡 Show answer
H = 100(tan45°+tan30°)/(tan45°−tan30°) = 100(1+1/√3)/(1−1/√3) = 100(√3+1)/(√3−1) = 100(2+√3) ≈ 373.2 m

📋 For Teachers & Parents

Topic: Height & Distance — Cloud Reflection Problem (Class 10 Mathematics)
Curriculum: NCERT Ch.8, CBSE, ICSE

Learning Objectives

  • Understand mirror reflection geometry (depth = height)
  • Apply angle of elevation and depression correctly
  • Set up two equations with shared horizontal distance
  • Solve simultaneous trigonometric equations
  • Derive and use the general formula H = h(tanβ+tanα)/(tanβ−tanα)

Common Errors

  • ✗ Forgetting reflection depth equals cloud height
  • ✗ Using h−60 for both triangles (reflection needs h+60)
  • ✗ Confusing elevation and depression angles
  • ✗ Adding angles instead of setting up separate triangles

Extension Activities

  • Verify with a real mirror: place object at distance h, image appears at depth h
  • Photograph a building reflected in water — measure angles
  • Research: How do surveyors use reflection principles?

The Bigger Picture

This same mathematics powers:

  • Maritime Navigation: Sextants measure angles of elevation to celestial bodies to find a ship's position at sea
  • Aerial Surveying: Aircraft altimeters combined with depression angles calculate ground feature distances
  • Coastal Engineering: Determining safe anchorage distances from shore-based observation posts
  • Astronomy: Parallax measurements use the same triangle geometry to measure distances to stars
Did you know? The ancient Greek astronomer Hipparchus (190-120 BC) created the first trigonometric table — a "chord table" — to calculate the distances to the Moon and Sun using exactly this kind of angle measurement from two different observation points.

NCERT / CBSE Alignment

Chapter 9 · Some Applications of Trigonometry · Exercise 9.1 · Q11

Learning Objectives

  • Understand angle of depression as measured below the horizontal
  • Apply alternate interior angles: angle of depression = angle of elevation
  • Use tan theta = opposite/adjacent to find horizontal distances
  • Calculate the difference between two distances to find separation
  • Work with exact values involving root 3

Common Errors

  • Adding the two angles (30 + 45 = 75) instead of using them separately
  • Calculating individual distances but forgetting to subtract them
  • Using sin or cos instead of tan for opposite/adjacent ratios
  • Confusing which angle corresponds to which ship (smaller angle = farther)

Prerequisites

  • Trigonometric ratios: sin, cos, tan
  • Special angles: 30, 45, 60 and their exact values
  • Alternate interior angles theorem
  • Working with surds (radical expressions)

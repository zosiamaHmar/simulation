Step 1 · The Scene
The Right Triangle in the Sky

Let A be the point on the ground where the string is tied.

Let B be the point directly below the kite (at ground level).

Let C be the kite itself.

Right Triangle ABC
AB = horizontal distance (unknown)
BC = height = 60 m (given)
AC = string length = ? (what we need)
∠CAB = 60° (angle with ground)
👁 Highlight the triangle on canvas →
Step 2 · The Key Insight
Which Trig Ratio to Use?

We know:

  • Angle with ground = 60°
  • Opposite side to this angle = height = 60 m
  • Hypotenuse = string length (unknown)

The ratio connecting opposite and hypotenuse is sine:

The Key Formula
sin(θ) = opposite / hypotenuse
sin(60°) = BC / AC = 60 / string
💡 Why sine and not tan? Click to see.
We need the hypotenuse (string). Sine connects opposite and hypotenuse. Tangent connects opposite and adjacent — that would give us the horizontal distance, not the string length!
Step 3 · The Calculation
Solving for String Length

From sin(60°) = 60 / AC:

Rearrange for AC
AC = 60 / sin(60°)

We know sin(60°) = √3/2:

AC = 60 / (√3/2)
AC = 60 × 2/√3
AC = 120/√3
AC = 120√3/3
AC = 40√3 metres

Therefore:

Final Answer
String length = 40√3 metres
≈ 69.28 metres
Do NOT use tan(60°) here — that gives the horizontal distance (20√3 ≈ 34.64 m), not the string length!
Step 4 · Verification
Does 40√3 Make Sense?
Length of String
40√3 m
≈ 69.28 metres
You guessed: -- · Actual: 40√3 m

Check: If string = 40√3 m

  • sin(60°) = 60 / (40√3) = 60√3 / 120 = √3/2
  • But sin(60°) = √3/2 ✓
  • Horizontal distance = 40√3 × cos(60°) = 40√3 × 0.5 = 20√3 ≈ 34.64 m ✓
  • Check: tan(60°) = 60 / (20√3) = 3/√3 = √3 ✓

Phase 5 · Master
Three Variants to Test Yourself
Variant 1 · Different Angle

Kite at 60 m height, string angle = 30°. Find string length.

💡 Show solution
String = 60 / sin(30°) = 60 / 0.5 = 120 m. Much longer! Lower angle = longer string needed.
Variant 2 · Different Height

String angle = 45°, string length = 80 m. Find kite height.

💡 Show solution
h = 80 × sin(45°) = 80 × 1/√2 = 40√2 ≈ 56.57 m
Variant 3 · Real World

A drone is flying with a tether at 75° angle. The tether is 100 m long. How high is the drone?

💡 Show answer
h = 100 × sin(75°) ≈ 100 × 0.966 = 96.6 m

📋 For Teachers & Parents

Topic: Height & Distance — Kite Problem (Class 10 Mathematics)
Curriculum: NCERT Ch.8, CBSE, ICSE

Learning Objectives

  • Visualize angle of elevation in a right triangle
  • Identify opposite side and hypotenuse correctly
  • Apply sin(θ) = opposite/hypotenuse
  • Work with special angle values (sin 60° = √3/2)
  • Verify answer using multiple trig ratios

Common Errors

  • ✗ Using tan instead of sin (finds horizontal distance)
  • ✗ Forgetting sin(60°) = √3/2, using 1/2 instead
  • ✗ Not rationalizing (leaving answer as 120/√3)

Extension Activities

  • Fly a real kite and estimate the string angle using a protractor
  • Measure the string length and verify with trigonometry
  • Research: How do kite festivals use geometry for safety?

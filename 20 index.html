Teacher & Parent Notes

Topic: Reciprocal Angles of Elevation (Class 10)
NCERT: Ch. 9, Exercise 9.1
Concepts: Two right triangles sharing a common base, tan ratios, simultaneous equations

Learning Objectives

  • Visualize two observers at different bases looking at each other's tops
  • Apply tan θ in two separate right triangles with common base
  • Set up and solve two equations with two unknowns
  • Recognize that the same distance appears in both triangles

Common Errors

  • Using different distances for the two triangles
  • Adding angles (60°+30°=90°) instead of using tan separately
  • Forgetting that both observers stand on the same ground level
  • Using sin/cos when tan is the direct tool
  • Not recognizing the common base as the key link

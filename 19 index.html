Teacher & Parent Notes

Topic: Heights & Distances (Class 10)
NCERT: Ch. 9, Exercise 9.1
Concepts: Angle of elevation, tangent ratios, simultaneous equations

Learning Objectives

  • Visualize angle of elevation from a real-world scenario
  • Apply tan θ = opposite/adjacent in two right triangles
  • Set up and solve simultaneous equations from equal heights
  • Verify solution using back-substitution

Common Errors

  • Adding angles (60° + 30° = 90°) instead of using tan ratios
  • Using sin or cos instead of tan for elevation problems
  • Forgetting that both pillars have EQUAL height
  • Incorrectly assigning x to the wrong pillar

Prerequisites

  • Values of tan 30°, tan 45°, tan 60°
  • Solving linear equations in one variable
  • Understanding of similar triangles

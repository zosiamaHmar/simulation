Teacher & Parent Notes

Topic: Heights & Distances (Trigonometry)

Alignment: NCERT Class 10, Ch. 9

Learning Objectives:

  • Understand angle of elevation from real-world observation
  • Apply tan θ = opposite/adjacent in two connected right triangles
  • Recognize that two triangles share the same base (horizontal distance)
  • Calculate tower height by subtracting building height from total

Common Errors:

  • Using sin instead of tan (forgetting adjacent = horizontal distance)
  • Adding the two angles instead of using them separately
  • Forgetting to subtract building height at the end

Real-World Connection: Surveyors use this exact method to measure inaccessible heights — tree height, mountain peaks, building floors.
